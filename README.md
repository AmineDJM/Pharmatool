# Algeria Pharma Market Intelligence App

## Lancer en local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déploiement Streamlit Cloud
1. Créer un repo GitHub privé.
2. Uploader `app.py`, `market_engine.py`, `requirements.txt` et le dossier `data/`.
3. Aller sur Streamlit Community Cloud.
4. New app → sélectionner le repo → `app.py`.
5. Deploy.
